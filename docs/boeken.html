    • https://bookdown.org/csgillespie/efficientR/

    • https://geocompr.robinlovelace.net/

    • The book is designed primarily for R users who want to improve their programming skills and understanding of the language.
    • https://adv-r.hadley.nz/

    • Nuttig als je met Git of GitHub aan de slag gaat.
    • https://happygitwithr.com/
  • Hands on programming with R - Garett Grolemund
    • “I wrote it for non-programmers to provide a friendly introduction to the R language. - Garret Grolemund”
    • https://rstudio-education.github.io/hopr/
  • R Markdown: The Definitive Guide - Yihui Xie, J.J. Allaire, Garrett Grolemund
    • Uitgebreide handleiding voor R Markdown
    • https://bookdown.org/yihui/rmarkdown/
  • bookdown: Authoring Books and Technical Documents with R Markdown - Yihui Xie
    • “This short book introduces an R package, bookdown, to change your workflow of writing books.”
    • https://bookdown.org/yihui/bookdown/
  • Statistical Inference via Data Science: A ModernDive into R and the Tidyverse - Chester Ismay and Albert Y. Kim
    • “Help! I’m new to R and RStudio and I need to learn them! What do I do?” If you’re asking yourself this, this book is for you.
    • https://moderndive.com/
  • R Graphics Cookbook, 2nd edition - Winston Chang
    • “a practical guide that provides more than 150 recipes to help you generate high-quality graphs quickly”
    • https://r-graphics.org/
  • ggplot2: elegant graphics for data analysis - Hadley Wickham, Danielle Navarro, and Thomas Lin Pedersen
    • “it’s primary focus is explaining the Grammar of Graphics that ggplot2 uses, and describing the full details.”
    • https://ggplot2-book.org
  • An Introduction to Statistical Learning with Applications in R - Gareth James, Daniela Witten, Trevor Hastie and Robert Tibshirani - 2018
    • Introductie in allerlei methodes voor Statistical learning (Machine learning)
    • http://faculty.marshall.usc.edu/gareth-james/ISL/
    • Zie ook de bijbehorende gratis cursus op de Stanford pagina https://lagunita.stanford.edu/courses/HumanitiesSciences/StatLearning/Winter2016/about
  • Forecasting: Principles and Practice - Rob J. Hyndman, George Athanasopoulos - 2020

    • https://otexts.com/fpp3/
